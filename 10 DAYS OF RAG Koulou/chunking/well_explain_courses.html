Les 7 Méthodes
de Text Splitting

Le but : Sortir la grosse artillerie.
Au lieu de bricoler nos fonctions, on importe. Une boîte à outils géante pour l'IA. Fini de réinventer la roue.
01
straighten

Taille Fixe (Pro)

Coupe précisément tous les X caractères. Version professionnelle codée par des experts. Rapide et sans bugs.

02
content_copy

Chevauchement

Ajoute chunk_overlap. Comme des photocopies intelligentes, LangChain gère les calculs pour la continuité.

03
format_quote

Sémantique (Pro)

Coupe uniquement aux points (.). Transforme le texte en une liste de phrases claires. Simple mais puissant.

04
account_tree

Hiérarchique / MD

Coupe selon les titres (#, ###). Colle une étiquette sur chaque morceau (ex: "Chapitre 2"). Vital pour l'organisation.

05
autorenew

Récursif (Standard Ind.)

Le standard de l'industrie pour le RAG. Essaie de garder les paragraphes ensemble avec 50 caractères de sécurité. Le meilleur compromis jamais inventé.

06
smart_toy

Adaptatif (Pro)

Logique intelligente : si HTML -> fais ça, si texte -> fais ça. Assemble des blocs selon le contexte.

07
code

HTML Structurel

Utilise HTMLHeaderTextSplitter. Comprend H1, H2, H3 parfaitement sans se mélanger les pinceaux.
